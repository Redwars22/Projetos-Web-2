COMO FUNCIONA E COMO USAR O AFTERMATH?

O Aftermath é um editor HTML/CSS online perfeito para protótipos e testar códigos. Funciona em computadores e tablets. Foi testado nos namevadores Opera e Brave, mas possivelmente funciona em todos os outros browsers baseados no Chromium.

COMO PREVISUALIZAR O CÓDIGO HTML?

Basta clicar no botão PREVIEW para obter uma prévia do teu código. Ela aparecerá do lado direito da tela.

book

PESQUISAR NA DOCUMENTAÇÃO HTML E CSS

Precisa pesquisar alguma coisa sobre o HTML e CSS? Não precisa de abrir uma nova guia e pesquisá-lo manualmente. Basta digitar o que deseja pesquisar no campo na parte superior do aplicativo e clicar em PESQUISAR. Uma nova guia será aberta no seu navegador com os resultados de busca do Google.

save

SALVAR E ABRIR PROJETOS

Você pode salvar o código para continuar trabalhando nele mais tarde, basta apenas clicar no botão salvar> no menu superior. Para continuar de onde parou posteriormente, clique no botão abrir. Clicar no botão novo irá deletar tudo o que estiver salvo e criar um novo projeto. Tenha certeza do que está fazendo ao selecionar esta opção, pois uma vez apagados os dados, isso não pode ser desfeito. O Aftermath usa o LocalStorage para salvar os dados e, portanto, para continuar trabalhando no projeto mais tarde, você deverá usar o mesmo dispositivo e navegador que foi utilizado para salvá-lo.

EM QUE PLATAFORMAS O AFTERMATH FUNCIONA?

O Aftermath funciona em praticamente qualquer plataforma, mas funciona melhor em computadores e tables. Ele foi testado no Opera e Brave, mas deve funcionar em qualquer navegador baseado no Chromium.
