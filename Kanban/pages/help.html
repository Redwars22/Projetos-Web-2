COMO USAR O PROJECT KANBAN?

  1. SALVAR E DELETAR: para salvar os dados basta clicar no ícone de disco rígido localizado no canto superior direito da tela. Para deletar tudo, clique no ícone da lixeira. OBSERVAÇÕES: Não se esqueça de salvar periodicamente as alterações feitas para não perdê-las! É importante ressaltar também que uma vez deletados os dados não é possível recuperá-los!
  2. CAMPOS TO-DO, DOING E DONE: no campo TO-DO você pode colocar as tarefas que devem ser feitas, no campo DOING as tarefas que estão em execução e, finalmente, no campo DONE você colocará as tarefas que já estão concluídas.
  3. CÓDIGO FONTE: o código fonte desta aplicação está disponível integralmente no GitHub. Você pode acessá-lo, baixá-lo, distribuí-lo e modificá-lo livremente.
